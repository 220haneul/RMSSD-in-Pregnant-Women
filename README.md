## Feature Description

### Electronic Medical Record (EMR) Features

- **Age**: Age of the participant.
- **Height**: Height of the participant.
- **Weight**: Weight of the participant.
- **Body mass index (BMI)**: Body mass index of the participant, representing body composition based on height and weight.
- **Gestational age**: Gestational age of the participant at the time of data collection, expressed in weeks.

### Wearable Sensor Features

- **Heart rate**: Heart rate measured by the wearable device at each time point.
- **Sleep score**: Sleep-related score provided by the wearable device, representing the participant's sleep status.
- **Step count**: Number of steps recorded by the wearable device for the corresponding time interval.
- **SpO2**: Peripheral oxygen saturation measured by the wearable device.
